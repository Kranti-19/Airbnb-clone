# 🏡 WanderLust – AI Powered Airbnb Clone

A full-stack accommodation booking platform inspired by Airbnb, enhanced with **AI-powered smart search and recommendations**.

## 🚀 Key Features

### 🔍 Smart Search Engine

* Search using natural language (e.g., *"rooms under 3000 in Pune"*)
* Automatically detects:

  * Price
  * Category
  * Location

### 🧠 AI Chat Assistant

* Suggests listings based on user queries
* Filters by price + category dynamically
* Improves user experience with conversational UI

### 🏷️ Category-Based Filtering

* Mountains 🏔️
* Farms 🌾
* Pools 🏊
* Castles 🏰
* Camping ⛺

### 🔐 Authentication System

* Secure login/signup using Passport.js
* Session-based authentication

### 📝 Listing Management

* Create, edit, delete listings
* Upload images via Cloudinary

### 💬 Reviews & Ratings

* Add and manage reviews
* Linked to listings

### 💰 Pricing UX

* Tax toggle feature
* Clean price display
